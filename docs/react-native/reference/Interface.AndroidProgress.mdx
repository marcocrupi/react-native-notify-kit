## Properties

### current?

> `optional` **current**: `number`

Defined in: [packages/react-native/src/types/NotificationAndroid.ts:837](https://github.com/marcocrupi/react-native-notify-kit/blob/main/packages/react-native/src/types/NotificationAndroid.ts#L837)

The current progress value.

E.g. setting to `4` with a `max` value of `10` would set a fixed progress bar on the notification at 40% complete.

***

### indeterminate?

> `optional` **indeterminate**: `boolean`

Defined in: [packages/react-native/src/types/NotificationAndroid.ts:845](https://github.com/marcocrupi/react-native-notify-kit/blob/main/packages/react-native/src/types/NotificationAndroid.ts#L845)

If `true`, overrides the `max` and `current` values and displays an unknown progress style. Useful when you have no
knowledge of a tasks completion state.

Defaults to `false`.

***

### max?

> `optional` **max**: `number`

Defined in: [packages/react-native/src/types/NotificationAndroid.ts:830](https://github.com/marcocrupi/react-native-notify-kit/blob/main/packages/react-native/src/types/NotificationAndroid.ts#L830)

The maximum progress number. E.g `10`.

Must be greater than the `current` value.
